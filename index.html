O que é HTML

HeyperText Markup Language HTML traduzindo Linguagem de Marcação de Hipertexto, é utilizado para realizar marcações atraves do que chamamos de tags que transformam a forma como interagimos e visualizamos o que foi escrito anteriormente, com o HTML5 que é o mais atual, podemos reproduzir audio ou video na propria pagina web, e realizar o que é obvio por levar o nome à linguagem, ancoragem de links que te levaram a navegar na rede!

O que é uma pagina Web

Uma pagina web criada através do auxilio da linguagem HTML é nada mais que uma soma de arquivos que é lida e interpretada pelos navegadores,por exemplo Safari, Mozila, Google Chrome e Edge, que organiza e projetam nos nossos monitores aquelas instruções definidas a partir do HTML, CSS, JS entre outros.

O que é um Server-side

O Server-side em tradução literal quer dizer, lado do servidor, podemos usar como exemplo mais comum a ilutração de um Bando de Dados, o BD seria um caso de uso de liguagens em server-side.

O que é um Client-side

Contrário ao Server-side o Client-side, funciona do lado que quem usa o sistema, nosso "usuário", através da linguagem mais conhecida nos dias de hoje que é o JavaScript, operando manipulações de funções, telas ou quaisquer ações que execute no navegador do usuário
